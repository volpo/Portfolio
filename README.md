# Portfolio Front End – Alejandro Volponi

Este proyecto es un **portfolio personal** desarrollado para la materia de **Front End**.  
El objetivo es aplicar conceptos de **HTML semántico**, **CSS (Flexbox y Grid)** y **diseño responsive**.

---

## Contenido del sitio

El portfolio está dividido en las siguientes secciones:

- **Inicio**: presentación principal, título y llamado a la acción.
- **Sobre mí**: breve descripción personal, intereses y skills principales.
- **Proyectos**: listado de trabajos/prácticos con descripción y links (GitHub / demo).
- **Contacto**: datos de contacto y enlaces a LinkedIn y GitHub.

---

## Tecnologías utilizadas

- **HTML5** (estructura y semántica)
- **CSS3**
  - Flexbox
  - Grid
  - Variables CSS
  - Diseño responsive
- (Opcional) **Git & GitHub** para control de versiones

---


## Cómo ver el proyecto

La versión deployada del portfolio se puede ver online en:

https://alejandro-volponi-portfolio.vercel.app/

Si se desea ejecutar de forma local:

1. Descargar o clonar este repositorio.
2. Abrir el archivo `index.html` en cualquier navegador web moderno.
3. Verificar que `styles.css` esté en la misma carpeta que `index.html`.
